Squarespace, Wix, Format and Cargo rent you a page and keep the keys. gesso hands you plain files in a GitHub repo you own. here is what that adds up to.

  • $0

    $0 forever

    forever. no plan, no upsell. the others charge $16 to $49 a month.

  • 100%

    100% yours

    yours. the repo, the images, every page is a plain file you hold.

  • 4+

    4 or more hosts

    hosts you can move to. GitHub Pages, Netlify, Cloudflare, your own server.

  • 0

    0 lines of code

    lines of code. drag photos in, publish. if gesso vanished your site keeps running.
